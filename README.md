# Responsive Portfolio Website

# Lab Title
**Responsive One-Page Portfolio using Flexbox, CSS Grid, Media Queries, and Animations**

# Description
This project is a one-page responsive portfolio website designed to showcase personal information, projects, skills, and contact details.  
It demonstrates modern frontend techniques including:
- *Flexbox** for alignment and spacing
- *CSS Grid** for project layout
- *Media Queries** for mobile-first responsiveness
- *Relative units** (`rem`, `em`, `%`, `vw`, `vh`) for scalable spacing and font sizing
- *Hover effects, transitions, and transforms** for interactivity
- *Keyframe animations** for polished visual appeal

The site adapts seamlessly across desktop, tablet, and mobile screens, ensuring a smooth user experience.

---

 Breakpoints Tested
- Desktop (≥ 1024px)
- Full layout with Flexbox and Grid.
- Navigation links displayed in a row.
- About section side-by-side (image + text).
- Tablet (~768px)
- Navigation links remain horizontal but text sizes scale down.
- About section stacks vertically for better readability.
- Mobile (≤ 480px)
- Navigation links stack vertically.
- Skills list centered.
- Font sizes and spacing adjusted for small screens.
- Interactive elements
- Hover over navigation links, project cards, and buttons to see transitions.
- Hero heading animates with a fade-in effect on load.
